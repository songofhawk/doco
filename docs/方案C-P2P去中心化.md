# 方案 C：P2P 去中心化 — WebRTC + 分布式存储

## 概述

使用 y-webrtc 实现客户端之间的直接 P2P 同步，配合 y-indexeddb 本地持久化，可选 IPFS/OrbitDB 做共享存储层。探索区块链思想（DHT 节点发现、内容寻址）在协同编辑中的应用。

## 架构

```
┌─────────────────────────────────────────────────────────┐
│                    前端（浏览器）                         │
├─────────────────────────────────────────────────────────┤
│  Yjs (CRDT)                                             │
│    ├─ y-webrtc（在线 P2P 同步）                          │
│    ├─ y-indexeddb（离线缓存）                            │
│    └─ y-websocket（可选：降级备选）                      │
├─────────────────────────────────────────────────────────┤
│  OrbitDB（可选：元数据去中心化同步）                      │
│    └─ IPFS + Libp2p Pubsub                              │
├─────────────────────────────────────────────────────────┤
│  Service Worker（离线支持）                              │
└─────────────────────────────────────────────────────────┘
```
