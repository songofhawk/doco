# 电子表格

Doco 支持两种不同形态的电子表格：

- **独立电子表格文件**：从知识库或文件夹的“+”菜单选择“新建电子表格”。页面主体直接是全尺寸表格，不经过 ProseMirror，也不显示文档正文、字数、目录或文档设置。
- **文档内嵌电子表格**：在普通文档中输入 `/电子表格`、`/spreadsheet`、`/excel` 或 `/dzbg` 插入。它是文档正文中的一个内容块。

独立电子表格的数据存储在 `Y.Map('spreadsheet')`；内嵌电子表格的数据存储在 `spreadsheetBlock` 节点属性中。两者共用表格交互组件，但具有不同的文件语义和持久化结构。

## 功能

- 单元格与范围选择，支持鼠标拖选、Shift 扩选和全选
- 键盘导航：方向键、Tab、Shift + Tab、Enter、Delete / Backspace
- 复制、剪切、粘贴 TSV/CSV 数据
- 公式栏和单元格内编辑
- 粗体、斜体、下划线、文字颜色、填充色、对齐
- 常规、数字、百分比、人民币、日期格式
- 合并与取消合并单元格
- 增删行列、追加行、拖动调整列宽
- 按当前列升序或降序排列
- 多列文本筛选
- 冻结到当前行或当前列
- CSV / TSV 导入和 CSV 导出
- 状态栏显示选区数量与数值求和
- 通过 IndexedDB、Yjs、Hocuspocus 和后端快照链路同步

## 公式

公式以 `=` 开头，支持单元格引用、范围、括号、字符串和以下运算：

- 算术：`+`、`-`、`*`、`/`、`^`
- 文本连接：`&`
- 比较：`=`、`>`、`<`、`>=`、`<=`、`<>`、`!=`
- 函数：`SUM`、`AVERAGE` / `AVG`、`MIN`、`MAX`、`COUNT`、`COUNTA`、`IF`、`ROUND`、`ABS`、`CONCAT`

示例：

```text
=SUM(A1:A10)
=IF(B2>=60,"通过","未通过")
=ROUND(AVERAGE(C2:C20), 2)
=A1&"-"&B1
```

循环引用显示为 `#CYCLE!`，除零显示为 `#DIV/0!`，其他公式错误显示为 `#ERROR!`。

## 导入导出

- 块内导入支持 `.csv`、`.tsv`、`text/csv` 和 `text/tab-separated-values`。
- 块内 CSV 导出的是当前计算结果。
- 普通文档导出 Markdown 时，内嵌电子表格降级为 `csv` 围栏代码块。公式文本会保留，但格式、筛选、冻结、合并和列宽无法由 Markdown/CSV 表达。
- Markdown 中的 `csv` 围栏代码块再次导入 Doco 时会恢复为电子表格块。
